BIO


I am an enterprise level, front-end Angular developer with proven success in building dynamic, fully responsive web applications in an Agile environment. Collaborating with diverse teams, including offshore developers, creative designers, product owners, scrum masters, project managers, back-end Java developers, architectural teams, and business leaders has given me tremendous experience in the Orlando area.

After spending 2 years as an Angular developer for Disney Resort Technology, in March 2020 I started as an Angular developer for Universal Studios Orlando. My team and I update, debug, and prevent defects on the Universal Studios Orlando and Universal Studios Hollywood, public-facing websites. I enjoy collaborating with other developers, improving site accessibility, implementing a complex, multi-tiered CMS into Angular codebase, partnering with QA testers and troubleshooting defects, while continuously expending my skills and knowledge base with Pluralsight in my personal time.

Other skills include HTML5, CSS3/Less/Sass, Flexbox, GRID, JavaScript/Typescript, jQuery, Bootstrap, HTTPS requests to restful API's, retrieving JSON data, AJAX calls, Selenium Webdriver, Real Time Cloud Database via Firebase, Heroku, version control via GitHub, managing local & session storage, code coverage/unit tests, Node.js, Express.js, MVC frameworks, User Authentication, Algorithms, and Design Patterns.
